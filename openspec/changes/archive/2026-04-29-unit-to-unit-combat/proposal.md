# Proposal: 1:1 Unit Combat System

## Why
현재 전투는 리더가 적 무리와 충돌 시 전체 숫자가 한 번에 깎이는 방식이라 시각적으로 밋밋함. 카운트 마스터처럼 아군 유닛과 적군 유닛이 실제로 부딪혀서 1:1로 사라지는 연출이 필요함.

## What Changes
- `EnemyMinion.cs`: 개별 적 유닛에 충돌 감지 로직 추가.
- `EnemyGroup.cs`: 무리 전체의 충돌체 제거, 개별 미니언들의 생성 및 관리 역할로 변경.
- `CrowdController.cs`: 특정 유닛이 적과 부딪혔을 때 해당 유닛만 정밀하게 제거하는 기능 강화.
- 시각 효과: 충돌 시 작은 연기나 파편 이펙트(선택 사항) 추가 가능.

## Impact
- 전투 장면이 훨씬 역동적이고 실제 병력끼리 싸우는 느낌을 줌.
- 유닛 하나하나의 생존이 중요해지는 긴장감 조성.
